# 프로젝트 개요
영화를 예매하는 영화관 프로젝트
* **개발 목적**: Docker, Jenkins, AWS를 활용한 CI/CD 구축 및 AWS를 통한 배포
* **개발 인원**: 개인 프로젝트  
* **개발 기간**: 1월 25일 ~ 현재 진행 중

# 사용 기술
* Spring boot
* JAVA
* JPA
* Spring Security
* MariaDB

# 주요 기능
* 영화 예매 및 취소 기능
* 댓글 작성 기능
* 로그인

* 영화 예매 좌석과 위치가 고정된것이 아니라 관리자가 설정할 수 있도록 변경

* Docker와 Jenkins를 활용한 CI/CD 구축
* AWS EC2를 활용한 배포
