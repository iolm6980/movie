# 프로젝트 개요
스프링부트의 기본적인 사용방법을 복습하고
CI/CD 구축 및 AWS 연습을 위한 영화관 프로젝트

* **개발 인원**: 개인 프로젝트  
* **개발 기간**: 11월 3일 ~ 12월 23일

# 사용 기술
* Spring boot
* JAVA
* JPA
* Spring Security
* MariaDB

# 주요 기능
* 영화 예매 및 취소 기능
* 댓글 작성 기능
* 로그인
